# Supported Frameworks

PiPilot provides comprehensive support for three modern web and mobile development frameworks, each optimized for different use cases and deployment scenarios.

## Next.js

Build full-stack web applications with the React framework that provides hybrid static and server-side rendering.

- **Full-Stack Development**: Server-side rendering and API routes
- **Static Site Generation**: Optimized for performance and SEO
- **File-Based Routing**: Intuitive page structure
- **Built-in Optimization**: Automatic code splitting and image optimization

## Vite + React

Create fast, modern client-side applications with Vite's lightning-fast build tool and React's component-based architecture.

- **Rapid Development**: Instant hot module replacement
- **Modern Build Tool**: Optimized for speed and developer experience
- **Component Architecture**: Reusable UI components
- **Client-Side Rendering**: Fast initial page loads

## Expo

Develop cross-platform mobile applications for iOS and Android using React Native with Expo's managed workflow.

- **Cross-Platform**: Single codebase for iOS and Android
- **Managed Workflow**: Simplified development and deployment
- **Native Features**: Access to device hardware and APIs
- **Live Updates**: Push updates without app store approval

## Framework Selection Guide

Choose the right framework based on your project requirements:

| Framework | Use Case | Rendering | Deployment |
|-----------|----------|-----------|------------|
| Next.js | Full-stack web apps | Hybrid SSR/SSG | Vercel, Netlify |
| Vite+React | Client-side SPAs | Client-side | Any static host |
| Expo | Mobile apps | Native | App stores |

<Tip>
PiPilot's AI can help you choose the optimal framework for your project based on your requirements and automatically set up the initial project structure.
</Tip>
